Bootstrap 4 Reference

Hello world.

This page is pretty much the minimum configuration required to implement Bootstrap 4 on an HTML page, using a CDN.

The key elements in the head are the meta tag specifying viewport and the bootstrap.min.css stylesheet. The three javascript libraries at the bottom of the body are also needed.
