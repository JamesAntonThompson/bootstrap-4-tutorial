Hand-Built Menu Iteration 2

In this iteration we add a navbar-nav class to the ul element which formats the links

We also add text-uppercase and ml-auto classes

In the div above the ul, we add collapse and navbar-collapse

We also add an id of navbarResponsive which will be used in by the javascript

In the nav element, we add navbar and navbar-expand-lg which turns the list into a horizontal navbar and has it be expanded with the screen width is large

In the nav element, we also add navbar-light to set it to be dark colours and and fixed-top to have it fixed to the top of the viewport
