Padding, Display and Margins

Padding and Margins

The way margins work inside bootstrap is that the different sizes reflect differet orders of magnitude relative to the size of the Sass Map Variable $spacer, which has a default size the same as the font size. So,

0 = 0 x font

1 = 0.25 x font

2 = 0.5 x font

3 = 1.0 x font

4 = 1.5 x font

5 = 3 x font

Bootstrap pading and margin classes use the following nomenclature: t = top, b = bottom, l = left, r = right. x = left and right, y = top and bottom, blank = all
