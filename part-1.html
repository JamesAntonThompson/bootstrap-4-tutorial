3. Media Breakpoints

There are several media breakpoints defined in Bootstrap 4. The key ones are:

  • XL = min width:1200px
  • LG = min width:992px
  • MD = min width:768px
  • SM = min width:576px
  • xtra-sm = max width:575.98px
